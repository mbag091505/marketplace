# marketplace
# 🌍 Africa Marketplace

A responsive multi-page marketplace website built with **HTML and CSS** that connects buyers and sellers across Africa.

This platform allows users to explore products from different African countries and promotes local trade by helping sellers reach a wider audience.

---

## 🚀 Features

* 🏠 Home page with hero section and features
* 🛍️ Product listing page with responsive grid
* 📖 About page with mission, vision, and stats
* 📞 Contact page with form and business info
* 📱 Fully responsive design (mobile-friendly)
* 🎨 Clean and modern UI
* 🌍 Focus on African products and sellers

---

## 📂 Project Structure

```
africa-marketplace/
│
├── index.html        # Home page
├── products.html     # Products listing
├── about.html        # About page
├── contact.html      # Contact page
├── style.css         # Styling file
└── images/           # (Optional) local images
```

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)

## ▶️ How to Run

1. Download or clone the project
2. Open the folder
3. Double-click on `index.html`

OR

Open in browser:

```
http://localhost/index.html
```
